# BigToe AI

A sophisticated AI-powered image generation application with a focus on gallery management and user subscription systems.

## Key Features

-   **Generator**: Create AI images with custom prompts and parameters.
-   **Gallery**: Organize images, mark favorites, and filter by tags.
-   **Subscription**: Tiered access (Basic, Pro, Premium) with credit management.
-   **Profile**: Manage settings (Language, GDPR) and view history.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Set up environment: Create `.env` with `VITE_GOOGLE_API_KEY`.
4.  Run locally: `npm run dev`.

## Deployment

Deployable to Vercel. Ensure `VITE_GOOGLE_API_KEY` is set in the deployment environment variables.
